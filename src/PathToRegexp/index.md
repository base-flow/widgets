# PathToRegexp

## 介绍

Url路径模式匹配，集成自`path-to-regexp@8.3.0`

## 示例

```ts

import { PathToRegexp } from '@baseflow/widgets';

const { keys } = PathToRegexp.pathToRegexp(path);

```

| compile      | 参见：`path-to-regexp`  |

| parse        | 参见：`path-to-regexp`  |
